## Deskripsi Proyek
Dashboard ini dibuat untuk memvisualisasikan data analitik pengiriman, performa vendor, serta manajemen sisa stok. Antarmuka dirancang murni menggunakan teknologi web dasar agar ringan, modern (*Glassmorphism*), dan mudah disajikan (deploy).

## Fitur Utama
- **Dark & Light Mode**: Pergantian tema dinamis (*Theme Toggle*).
- **Responsif**: Tampilan adaptif untuk segala perangkat (Mobile, Tablet, Desktop) tanpa mengorbankan fungsionalitas.
- **Data-Driven Charts**: Visualisasi data interaktif menggunakan Chart.js (Grafik Sisa Stok, VPI, Performa Pengiriman, Jumlah Pesanan).
- **Optimasi Performa**: Pendekatan *Mobile-First* dengan implementasi *Hardware Acceleration* pada CSS demi animasi yang mulus.

## Tech Stack
Proyek ini dibangun tanpa framework besar (*zero-dependencies* selain Chart.js) untuk menjaga kecepatan dan kemudahan akses:
- **HTML5**
- **CSS3 (Vanilla)**
- **JavaScript (Vanilla ES6+)**
- **Chart.js**
